# 결제 모듈 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명: 패션 쇼핑 결제 모듈

### 사용하게 될 기술:  React

### 문서 작성자: 설동현

## 📝 고객 요구사항 정리
  - 신용카드 등록 기능
  - 테스트 URL 배포
  - 민감한 정보 보안 철저 및 뒷면 8자리 및 보안코드 숨김 처리

## 📋 기능 목록
- 카드 번호 input
  - `<CardNumberInput />`

- 만료 월/년(이중 입력, MM/YY)
  - `<CardExpiryInput />`

  - CVC(CVV) 숫자만 입력, 자리수 제한
  - 보안용 mask(숨김 처리)
    - `<CardCvcInput />`

- 카드 소유자 이름 입력
  - `<CardOwnerInput />`

- 카드 비밀번호 앞 2자리 입력
- 입력값 mask
  - `<CardPasswordInput />`
